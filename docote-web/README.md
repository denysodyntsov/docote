# DoCoTe Web MVP

DoCoTe Web is a web-first MVP for exploring how code changes, tickets, and existing documentation can be turned into clearer documentation follow-up, release guidance, and concrete output drafts.

## What it demonstrates

The current MVP demonstrates:
- repository / scope selection concepts
- mock analysis flow for PRs, branches, and commit ranges
- impacted documentation target identification
- documentation drift and release-readiness signals
- structured deliverables such as docs drafts and release-note drafts
- export/share-oriented output payloads
- a demo-friendly result experience with grouped sections and highlights

## Run locally

From this directory:

```bash
npm install
npm run dev
```

Then open:

- <http://localhost:3000>

## Suggested demo flow

1. Show the page hero and repo/scope framing
2. Point out the interactive demo form
3. Run the mock analysis
4. Review MVP highlights and demo narrative
5. Show impacted documentation targets
6. Show release readiness and review lane
7. Show structured deliverables
8. Show export/share

## Notes

This MVP is intentionally mock-driven and product-oriented. It is designed to validate workflow, framing, and output structure before deeper backend integrations are added.
