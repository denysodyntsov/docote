# MVP Demo Script

## Goal
Help present the DoCoTe web MVP in a short, coherent sequence.

## Suggested flow
1. Show repo/scope context
2. Run analysis
3. Point to MVP highlights
4. Show impacted documentation targets
5. Show release readiness and review lane
6. Show deliverables
7. Show export/share action

## Why this matters
A good demo flow helps the MVP land better than a raw feature tour.
