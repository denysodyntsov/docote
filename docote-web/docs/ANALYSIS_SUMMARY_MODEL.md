# Analysis Summary Model

## Purpose
DoCoTe should surface not only raw generated outputs, but also a compressed operational summary for users.

## Current step
Add a small model that derives:
- headline
- risk level
- update priority

from the analysis response and document impact severities.

## Why this matters
This makes the product more decision-oriented and less like a raw text generator.

## Next step
Surface the summary card in the interactive UI.
