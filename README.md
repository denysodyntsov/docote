# DoCoTe

DoCoTe is a documentation-first product for Jira-based software teams.

This repository currently contains:
- `docote-landing/` — landing page and early access form
- `docote-jira-app/` — first Jira app scaffold for generating:
  - technical implementation summary
  - documentation draft
  - release summary
- `DOCOTE_MVP_V1.md` — product MVP definition
- `DOCOTE_JIRA_PLUGIN_ARCHITECTURE_V1.md` — first architecture draft

## Current state
This is an early-stage product and implementation scaffold.

## Main idea
Turn Jira issues and code changes into usable technical documentation and release-ready summaries.
